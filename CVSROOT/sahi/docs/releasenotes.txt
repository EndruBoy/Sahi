Sahi Nightly Release 2005-12-26

Requirements
------------

* JRE 1.4 or above
* ant, log4j.jar and junit.jar if the module is to be rebuilt.

Changes
--------------------

* Fixed bug in handling cookies.
* Fixed bug in handling HTTPS POST requests.
* Added _call(fnCall()) method for delayed function calling.
